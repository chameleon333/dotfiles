# dotfiles

## Overview

This repository contains scripts to set macOS.

## Required

```shell
% sw_vers
ProductName:            macOS
ProductVersion:         14.7.6
BuildVersion:           23H626
```

```shell
% zsh --version
zsh 5.9 (x86_64-apple-darwin23.0)
```

## Install

Download installation materials.

```shell
% cd ~ && git clone https://github.com/chameleon333/dotfiles
```

Set macOS.

```shell
% cd ~/dotfiles && make
```
